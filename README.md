# Proyecto-Final-1400
Widget Flotante Pomodoro
********************************************************************************************************************************
El Widget Flotante Pomodoro es una herramienta visible en la pantalla del usuario que permite iniciar sesiones de trabajo, 
visualizar el tiempo restante y recibir notificaciones cuando el tiempo ha terminado.
Este proyecto aplica la tecnica "Pomodoro" para ayudar en la concentracion de tareas, permitiendo al usuario crear una lista
de tareas a desarrollar y definir el espacio de tiempo para dichas tareas. La tecnica "Pomodoro" divide en periodos de tiempo
de trabajo y periodos de tiempo de descanzo.
Esta aplicacion permitira al usuario crear su lista de tareas a desarrollar, el tiempo total destinado a esta sesion de estudio,
automaticamente dividira el tiempo y las tareas segun la tenica "Pomodoro" y mostrara en pantalla un semaforo de tiempo indicando
el estado de las tareas.
